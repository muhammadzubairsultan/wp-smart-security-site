WP Smart Security Documentation

Installation

  1. Upload plugin ZIP
  2. Activate plugin
  3. Go to WP Security → Dashboard

Activate Pro

  1. Purchase plugin on CodeCanyon
  2. Copy purchase code
  3. Paste into license field
  4. Activate

Features

  • Login attempt limiting
  • Custom login URL
  • Email security alerts
  • Auto updates (Pro)
